## 🤝 Beitragen

Kennst du eine gute Alternative die noch fehlt? Pull Requests sind willkommen!

1. Fork das Repository
2. Füge eine neue Karte in `alternatives.html` hinzu
3. Stelle sicher dass alle relevanten Tags gesetzt sind
4. Öffne einen Pull Request

## 📋 Selbst-Hosting Empfehlungen

Diese Übersicht entstand im Kontext eines **Debian Home-NAS/Servers**. Folgende Tools lassen sich besonders gut selbst hosten:

- **[Vaultwarden](https://github.com/dani-garcia/vaultwarden)** – Passwort-Manager (Docker)
- **[Immich](https://immich.app)** – Google Photos Ersatz (Docker)
- **[Seafile](https://www.seafile.com)** – File-Sync (Docker)
- **[Baikal](https://sabre.io/baikal/)** – CalDAV/CardDAV (Docker)
- **[Jellyfin](https://jellyfin.org/)** – Mediatheke für's eigene Heim (Docker oder Repo)
